## 운동 중독 플레이어
### 필요한 개념
* split

### 분석
split 함수를 사용해 입력 값을 분리하고, 각 값을 W와 R 변수에 할당한 후 toDouble 함수를 사용하여 문자열을 실수로 변환한다.

```kotlin
fun main() {
    // 입력
    val input = readLine()!!.split(" ")
    val W = input[0].toDouble()
    val R = input[1].toDouble()

    // 1RM 계산
    val answer = (W * (1 + R / 30)).toInt()

    // 결과
    println(answer)
}

```
